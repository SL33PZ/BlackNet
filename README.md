# Blacknet





#### Require
**OS**: Ubuntu
**Packages**: nodejs / npm
*Package install*: apt-get -y install npm nodejs


## Installation

Go to ***Release***, and download ***blacknet-x86_64.run***.
If the download is finished, execute *chmod +x blacknet-x86_64.run*, 
lastly execute *sudo ./blacknet-x86_64.run*.
